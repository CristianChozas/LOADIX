# LOADIX Backend

## Local setup

1. Start local infrastructure:
   - `docker compose up -d`
2. Copy backend env values from `backend/.env.example` into your shell or local env manager.
3. Run the backend once the application bootstrap is available.

## Local services

- PostgreSQL: `localhost:5432`
- Redis: `localhost:6379`

## Notes

This repository is still in reconstruction.
At this stage, this document defines the expected local environment before the backend bootstrap is ported.
